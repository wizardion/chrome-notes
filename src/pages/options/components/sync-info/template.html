Synchronize all data with Google Drive. It will be stored in one file on Google Drive, if this file is removed, the data will be lost on all synced devices.
To protect sensitive data, a secret key must be entered on each device, in which case the encryption will be enabled and the data will be encrypted using AES-GCM encryption. If this key is lost, it will be impossible to recover the data. Please remember that key and keep it secret.

In order you need to send the logs to a developer, you will need to print the logs and copy all details into a ticket.
print logs - prints all system logs.
clear logs - clears all system logs.
print cache - prints all values in cached storage.
clear cache - clears cached values.
print data - prints all database storage.
clear all data - clears all data including passphrase. Attention! This action is irreversible!

A) "Play" or "Pause" button.
  Press it to start or pause the timer.
B) "Reset time" button.
  This button will reset the time to: 00:00:00, and time tracking to: 15m.
C) "Add" button.
  This is a control button.
  When you click on it, a new timer appears.
  Tip: To see controls buttons, you should press "Ctrl" on you keyboard.
D) "Expand" button.
  Press it to expand timer and show the notes.
  Tip: You can also press "Tab" or "Shift + Tab" on your keyboard.
E) "Remove" button.
  This is a control button.
  Remember, all data including your notes on the selected timer will be removed.
  Tip: To see controls buttons, you should press "Ctrl" on you keyboard.
F) Started time.
  This option shows the time when the timer was started.
  Tip: To display this time, press "Ctrl + Alt" on you keyboard.
G) All Started time.
  This option shows the time for all timers when they were started.
  Tip: To display this, press "Ctrl + Alt + a" on you keyboard.
Enable Search: Note: if this setting is disabled, the search button will be hidden.
Sortable areas: Note: if this setting is disabled, sort areas will be hidden, but you can see them if you press the CTRL key on your keyboard.
Updated Date: Note: you can disable this setting, if there is no need to see when the note was updated.
